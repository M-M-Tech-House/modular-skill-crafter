# Modular Skill Crafter

Comprehensive guidelines and rules for creating, structuring, and maintaining agent skills that adhere to standard skill architecture.

## Overview

This repository serves as a reference implementation and guide for creating high-quality, modular skills for AI agents. It provides a structured approach to defining skills, rules, and best practices.

## Installation

To install this skill for your AI agent, use the following command:

```bash
npx skills add M-M-Tech-House/modular-skill-crafter
```

## Features

- **Standardized Structure**: Clear guidelines on directory hierarchy (`SKILL.md`, `rules/`, `examples/`).
- **Metadata Management**: Best practices for YAML frontmatter and semantic naming.
- **Rule Organization**: Strategies for breaking down complex rules into manageable, modular markdown files.
- **Content Formatting**: Templates for defining clear "Why", "Incorrect", and "Correct" rule examples.

## Usage

Once installed, this skill provides your agent with the knowledge to:

1. Create new skills from scratch following best practices.
2. Refactor existing skills for better modularity.
3. Validate skill structure and compliance.
4. Organize skill resources effectively.

For detailed documentation on specific rules, refer to the `rules/` directory within the installed skill.
